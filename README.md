## VF Template

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run prod
```

### Start development
Develop from the src/main.ts, documentation see [vf-engine](https://yunkc.gitee.io/docs/).
